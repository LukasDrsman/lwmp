# lwmp
A simple mp3 player, written in POSIX shell

## Dependencies
 - `mpg123`
## Usage
| command | effect | args |
|---------|--------|------|
|`lwmp`   |play album | `[album folder]`, `[volume percentile]` |
|`lwmp_stop`|stop lwmp | none |
|`lwmp_skip`|skip song | none |
|`lwmp_status`|print status | none |
